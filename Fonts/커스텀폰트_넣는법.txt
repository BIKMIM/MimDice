ㅁ 내 폰트 넣는 법

1. 원하는 한글 폰트 파일을 "와우의 Fonts 폴더가 아닌, 밈다이스 안의 Fonts 폴더"에 복사하세요.

   - 정확한 위치: 와우 설치 폴더\_retail_\Interface\AddOns\MimDice\Fonts
   - 지금 이 안내문이 들어있는 바로 이 폴더예요.
   - 와우 설치 폴더 바로 아래의 Fonts 폴더에 넣으면 안 돼요! 헷갈리지 마세요.

2. 와우 게임을 완전히 껐다가 다시 켜세요. (/reload 로는 새 파일을 인식하지 못해요)

3. 밈다이스 스킨 탭 > [폰트] 버튼 > 아래 입력칸에 파일 이름을 그대로 적고 [추가]를 누르세요.
   (예: 나눔스퀘어.ttf)

4. 폰트 목록에 그 이름 그대로 나타납니다.

ㅁ 되는 폰트 / 안 되는 폰트

1. 되는 것
   - .ttf (트루타입 폰트)
   - .otf (오픈타입 폰트)

2. 안 되는 것
   - .woff / .woff2 : 홈페이지용 웹폰트라서 게임에서 못 써요
   - .ttc : 여러 폰트가 한 파일에 묶인 모음 파일이라 못 써요
   - 가변(Variable) 폰트 : 파일이 열려도 글자가 이상하게 보일 수 있어요

3. 한글이 들어있는 폰트만 정상적으로 보여요.
   (영문 전용 폰트는 한글이 네모 또는 기본 글씨로 나와요)
